# Install PowerLink

This guide walks you through installing and configuring PowerLinks in your Agentix workspace.

## Installation Process

### Step 1: Choose Your PowerLink
Browse the [PowerLinks Catalogue](/user-guide/powerlinks/catalogue) and select the PowerLink that best fits your needs.

### Step 2: Install
1. Click **Install** on your chosen PowerLink
2. Review the PowerLink description and capabilities
3. Confirm the installation
4. Wait for the installation to complete

### Step 3: Initial Configuration
After installation, you'll be prompted to configure:
- **Basic settings**: Name, description and general behavior
- **Integration settings**: API keys, webhooks and external connections
- **Default parameters**: Standard values for common use cases

### Step 4: Verify Installation
1. Check that the PowerLink appears in your **Installed PowerLinks** list
2. Verify all configurations are correct
3. Test basic functionality if available
4. Review any setup warnings or recommendations

## PowerLink Management

### Updating PowerLinks
- PowerLinks receive automatic updates for bug fixes
- Feature updates require manual approval
- You'll be notified of available updates in the dashboard

### Uninstalling PowerLinks
1. Ensure no active workflows are using the PowerLink
2. Navigate to **Installed PowerLinks**
3. Click **Uninstall** on the PowerLink you want to remove
4. Confirm the uninstallation

## Troubleshooting

### Common Issues
- **Installation failed**: Check your internet connection and try again
- **Configuration errors**: Verify all required fields are completed
- **Integration problems**: Test API connections and credentials
- **Permission issues**: Ensure you have Manager role permissions

### Getting Help
- Review PowerLink-specific documentation
