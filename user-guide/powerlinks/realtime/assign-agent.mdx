# Assign Agents to Realtime

Configure AI and human agents to handle real-time customer conversations.

## Agent Types

### AI Agents
- Available 24/7 for immediate response
- Consistent performance and behavior
- Ideal for common questions and routing

### Human Agents
- Handle complex situations requiring empathy
- Provide personalized service and problem-solving
- Available during configured business hours

### Hybrid Configuration
- Start with AI agent for initial interaction
- Escalate to human agent based on triggers
- Maintain conversation context during handoff
