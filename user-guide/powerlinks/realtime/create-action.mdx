# Create Realtime Action

Learn how to create and configure a new Realtime action for live customer interactions.

## Step 1: Navigate to Actions

1. Go to **PowerLinks** â†’ **Actions** in your cPanel
2. Click **Create New Action**
3. Select **Realtime** from the PowerLink options

## Step 2: Basic Configuration

### Action Details
- **Name**: Give your action a descriptive name (e.g., "Customer Support Chat")
- **Description**: Explain what this action will accomplish
- **Tags**: Add tags for easy organization and filtering

### Channel Settings
- **Voice**: Enable voice conversations
- **Chat**: Enable text messaging
- **Video**: Enable video calls (optional)
- **Screen Share**: Allow screen sharing (optional)

## Step 3: Access and Triggers

### Public Access
- **Public URL**: Generate a public link for customer access
- **Embed Code**: Get code to embed in your website
- **Widget**: Configure a floating chat widget

### Trigger Conditions
- **Immediate**: Start conversation immediately when accessed
- **Scheduled**: Only available during specific hours
- **Conditional**: Based on customer data or behavior

## Step 4: Initial Settings

### Welcome Configuration
- **Welcome message**: First message customers see
- **Greeting style**: Formal, casual or custom tone
- **Language**: Primary language for interactions

### Queue Management
- **Queue handling**: What happens when agents are busy
- **Wait time messaging**: Updates for customers in queue
- **Callback options**: Allow customers to request callbacks
