</Note>

## Profile Settings

The Profile tab contains your organization's basic information and branding settings.

### Organization Profile

<AccordionGroup>
<Accordion title="Basic Information" icon="circle-info">
Configure your organization's essential details and contact information.

**Required Fields:**
- **Name** - Your organization's display name (e.g., "Agentix")
- **Address** - Physical business address (e.g., "Dubai")
- **Country** - Your organization's primary country (e.g., "United Arab Emirates")
- **About** - Description of your organization and business focus
</Accordion>

<Accordion title="Organization Logos" icon="image">
Upload your organization's branding assets for consistent visual identity.

**Logo Requirements:**
- **Light Logo** - For light theme backgrounds (PNG/SVG recommended)
- **Dark Logo** - For dark theme backgrounds (PNG/SVG recommended)  
- **Icon** - Square icon for compact displays (PNG/SVG, 1:1 aspect ratio)

**Specifications:**
- Maximum file size: 300kb per image
- Supported formats: PNG, SVG, JPG
- Recommended dimensions: 200x50px for logos, 64x64px for icons
</Accordion>
</AccordionGroup>

### Profile Configuration

<Steps>
<Step title="Enter Organization Details">
Fill in your organization name, address and country information in the respective fields.
</Step>

<Step title="Add Organization Description">
Provide a brief description of your organization in the "About" section to help users understand your business.
</Step>

<Step title="Upload Branding Assets">
Upload your light logo, dark logo and icon to maintain consistent branding across the platform.
</Step>

<Step title="Save Changes">
Click the save button to apply your organization profile updates.
</Step>
</Steps>

## Notifications Settings

The Notifications tab allows you to configure the communication providers that will be used to send notifications and alerts from your organization.

### Provider Configuration

Configure the communication providers that will handle different types of notifications from your Agentix organization.

<AccordionGroup>
<Accordion title="Email Server" icon="envelope">
Configure your email server settings for sending email notifications.

**Configuration Fields:**
- **Provider Dropdown** - Select email provider (Gmail, SMTP Server, etc.)
- **Username** - Email address for authentication (e.g., "slinkxy.notifications@gmail.com")
- **SMTP Server** - Server hostname for email delivery
- **Password** - Authentication password for the email account
- **SMTP Port** - Port number for SMTP connection (typically 587 or 465)

**Setup Instructions:**
1. Select your email provider from the dropdown menu
2. Enter your email username/address
3. Configure SMTP server settings
4. Provide the authentication password
5. Set the appropriate SMTP port
6. Test the connection to verify settings
</Accordion>

<Accordion title="SMS Provider" icon="mobile">
Configure your SMS service provider for sending text message notifications.

**Configuration Fields:**
- **SMS Provider Dropdown** - Select SMS service (SMS Global, Twilio, etc.)
- **API Key** - Your SMS provider's API authentication key
- **API Secret** - Secret key for API authentication
- **API URL** - SMS provider's API endpoint URL
- **Auth Token** - Additional authentication token if required

**Setup Instructions:**
1. Choose your SMS provider from the dropdown
2. Enter your API key from your SMS provider account
3. Add the API secret for authentication
4. Configure the API URL endpoint
5. Provide auth token if required by your provider
6. Test SMS delivery to verify configuration
</Accordion>

<Accordion title="WhatsApp Provider" icon="message">
Configure WhatsApp Business API for sending WhatsApp notifications.

**Configuration Fields:**
- **API URL** - WhatsApp Business API endpoint (e.g., "https://api.novomind-messaging.com/api/v1/")
- **Auth Token** - Authentication token for WhatsApp Business API access

**Setup Instructions:**
1. Obtain WhatsApp Business API access
2. Enter the API URL provided by your WhatsApp provider
3. Add your authentication token
4. Verify API connection
5. Test WhatsApp message delivery

**Requirements:**
- WhatsApp Business API account
- Verified business profile
- Approved message templates
</Accordion>
</AccordionGroup>

### Provider Setup Steps

<Steps>
<Step title="Select Communication Providers">
Choose which providers you want to configure: Email Server, SMS Provider and/or WhatsApp Provider.
</Step>

<Step title="Enter Provider Credentials">
Fill in the required authentication details for each selected provider.
</Step>

<Step title="Configure Server Settings">
Set up server URLs, ports and other technical configuration parameters.
</Step>

<Step title="Test Connections">
Verify that each provider is properly configured by sending test messages.
</Step>

<Step title="Save Configuration">
Click the "Save" button to apply your notification provider settings.
</Step>
</Steps>

## Access Organization Settings

To access your organization settings in the cPanel:

1. **Navigate to Settings** - Click the "Settings" option in the left sidebar of your cPanel
2. **Choose Tab** - Select either "Profile" or "Notifications" tab at the top
3. **Configure Settings** - Update your organization information or notification providers
4. **Save Changes** - Apply your settings by clicking the "Save" button

<Note>
Organization settings changes are applied immediately and affect all users in your organization.
</Note>
