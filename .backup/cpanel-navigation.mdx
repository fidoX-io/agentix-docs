# cPanel Navigation

The Agentix cPanel is designed to provide easy access to all features while maintaining a clean, intuitive interface.

## Main Navigation

### Dashboard
Your central hub showing:
- Active jobs and tasks overview
- System health and performance metrics
- Recent activity feed
- Quick action buttons

### PowerLinks
Manage your automation components:
- **Catalogue**: Browse and install new PowerLinks
- **Installed**: Configure your active PowerLinks
- **Actions**: Create and manage PowerLink actions

### Workflows
Design your customer journeys:
- **Design Studio**: Visual workflow builder
- **Templates**: Pre-built workflow templates
- **Active Workflows**: Monitor live workflows

### Jobs & Tasks
Operational management:
- **Active Jobs**: Currently running customer journeys
- **Task Queue**: Pending tasks requiring attention
- **History**: Completed jobs and their outcomes

### Tools & Sources
Configure your resources:
- **Knowledge Sources**: Manage knowledge bases and documentation
- **Tools**: Configure APIs and automation tools
- **MCP Tools**: Model Context Protocol integrations

### Monitoring
Track performance:
- **Real-time Dashboard**: Live metrics and alerts
- **Reports**: Historical performance analysis
- **Analytics**: Deep insights into customer interactions

### Settings
Organization configuration:
- **Organization Profile**: Company information and preferences
- **Branding**: Customize the interface for your brand
- **User Management**: Manage team members and permissions

## Quick Actions

The cPanel includes quick action buttons for common tasks:
- Create new workflow
- Install PowerLink
- Assign urgent task
- View system alerts

## Responsive Design

The interface adapts to different screen sizes:
- **Desktop**: Full feature access with expanded views
- **Tablet**: Optimized layout for touch navigation
- **Mobile**: Essential features for on-the-go access
